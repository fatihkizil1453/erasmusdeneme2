# Erasmus — Örnek Ana Sayfa

Bu proje, basit ve responsif bir ana sayfa şablonu sunar.

Dosyalar:
- `index.html` — Ana sayfa
- `styles.css` — Stil dosyası
- `script.js` — Basit etkileşimler (mobil menü, form doğrulama)

Nasıl çalıştırılır:
1. Bu klasördeki `index.html` dosyasını bir tarayıcıda açın.
2. Ya da PowerShell'de şu komutu kullanarak varsayılan tarayıcıda açabilirsiniz:

```powershell
Start-Process index.html
```

Özelleştirme önerileri:
- Renk paletini `styles.css` içindeki `--accent` değişkeniyle güncelleyin.
- Görselleri `hero-visual` içine gerçek resimlerle değiştirin.

İsterseniz tasarım stilini (renk, font, düzen) veya içerikleri değiştireyim.
